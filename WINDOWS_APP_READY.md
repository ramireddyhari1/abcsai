# 🎉 Anantapur Book Center - Windows Desktop App

## ✅ Installation Complete!

Your discount calculator is now available as a **Windows EXE application** that can be installed on any Windows computer.

---

## 📦 What You Have

### **Portable Executable** (No Installation Required)
- **Location:** `release-builds\discount-calculater-win32-x64\discount-calculater.exe`
- **Size:** ~200 MB (includes Electron framework)
- **Usage:** Double-click to run immediately
- **Portability:** Can be copied to USB drive or any folder

### **Installation Scripts**
1. **INSTALL.bat** - Professional Windows installer
   - Creates `Program Files` installation
   - Adds Desktop shortcut
   - Adds Start Menu shortcut
   - Creates Uninstall.bat for easy removal

2. **RUN.bat** - Quick test launcher
   - Launch the app directly from batch file

---

## 🚀 How to Use

### **Option 1: Portable (Fastest)**
```
1. Navigate to: release-builds\discount-calculater-win32-x64\
2. Double-click: discount-calculater.exe
3. App launches immediately!
```

### **Option 2: Install (Professional)**
```
1. Right-click INSTALL.bat
2. Select "Run as Administrator"
3. Follow prompts
4. App appears on Desktop and Start Menu
```

### **Option 3: Quick Test**
```
1. Double-click RUN.bat
2. App launches in test mode
```

- Installs to: `C:\Program Files\Anantapur Book Center\Discount Calculator\`

- Creates uninstaller at: `Program Files\Anantapur Book Center\Discount Calculator\Uninstall.bat`

## 🔄 Updates & Maintenance

**To update the app:**
1. Rebuild using: `npm run package:win`
2. Copy new files to installation folder
3. Or reinstall using INSTALL.bat

**To uninstall:**
- **Installed version:** Run `Uninstall.bat` from installation folder
- **Portable version:** Simply delete the folder
