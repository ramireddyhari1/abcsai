# Anantapur Book Center - Discount Calculator
## Windows Desktop Application

Professional discount calculator application for Anantapur Book Center.

---

## Installation Options

### **Option 1: Easy Installation (Recommended)**

1. **Download the installer**
   - Double-click `INSTALL.bat`
   - Click "Yes" when Windows asks for admin permission
   - Follow the on-screen instructions

2. **The installer will:**
   - Install the app to `Program Files`
   - Create a desktop shortcut
   - Create a Start Menu shortcut
   - Install an uninstaller

3. **Launch the app:**
   - Click the desktop shortcut
   - Or search "Anantapur Book Center" in Windows Start Menu

---

### **Option 2: Portable Executable (No Installation)**

The standalone executable is located at:
```
release-builds\discount-calculater-win32-x64\discount-calculater.exe
```

Simply:
1. Navigate to `release-builds\discount-calculater-win32-x64\`
2. Double-click `discount-calculater.exe` to run

**Note:** This folder contains all necessary files. You can copy the entire `discount-calculater-win32-x64` folder anywhere on your computer and run the EXE from there.

## Uninstalling

### If you used INSTALL.bat:
1. Open `Program Files\Anantapur Book Center\Discount Calculator\`
2. Double-click `Uninstall.bat`
3. Click "Yes" for admin permission
4. The application and shortcuts will be removed

### If you're using the portable version:
Simply delete the folder. No installation files to clean up.
